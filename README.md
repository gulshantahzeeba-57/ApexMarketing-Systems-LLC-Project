# Apex Marketing Systems LLC - Business Website Clone 

A modern, high-performance, and visually stunning corporate website built from scratch. This project is a front-end clone of the **Apex Marketing Systems LLC** website, designed to practice advanced CSS layouts, clean typography, and pixel-perfect responsiveness.

---

## Features

* **Fully Responsive Design:** Seamless user experience across mobile, tablet, and desktop screens.
* **Modern Aesthetic:** Deep obsidian dark background combined with sleek corporate blue accent elements.
* **Navigation Flow:** Clean navigation links including a non-intrusive, styled brand logo linking back to home without styling breaks.
* **Interactive Elements:** Smooth hover transitions on cards, buttons, and text-based project links.
* **Optimized Code:** Pure HTML5 and semantic CSS3 with zero reliance on heavy JavaScript or external frameworks.

---

##  Technologies Used

* **HTML5:** Semantic markup and structured page layouts.
* **CSS3:** Custom properties (variables), Flexbox, CSS Grid, and custom animations.
* **Google Fonts:** Clean corporate typography (using *Space Grotesk* and *Plus Jakarta Sans*).
